# semaforo-inteligente-arduino
El proyecto consiste en diseñar un semáforo inteligente capaz de cambiar el tiempo de luz verde dependiendo si existen vehículos detectados por un sensor de tráfico.
# Semáforo Inteligente con Arduino

## Descripción
Proyecto de sistema digital secuencial utilizando Arduino y sensor de tráfico.

## Funciones
- Control de luces
- Sensor de tráfico
- Cambio automático de estados

## Materiales
- Arduino UNO
- LEDs
- Resistencias
- Sensor HC-SR04
